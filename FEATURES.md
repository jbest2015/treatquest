# Treat Quest Features

## Current Version: 4.1

## Core Game
- Harley and Shanti with distinct looks
- Auto-running arcade mode
- Treat collection with scoring
- Collision avoidance

## Environment
- Real Tampa weather (Open-Meteo API)
- Dynamic time of day (sunrise/day/sunset/night)
- Hills background
- Fire hydrant
- Snoopy-style doghouse
- Puddles (slow dogs)

## Creatures
- Butterflies (day)
- Fireflies (night)
- Birds (V-formations)
- Squirrel with acorn bonus
- Beach ball (physics)

## Special
- Golden treats (5 points)
- Dog sleep mode (night)
- Live scoreboard
